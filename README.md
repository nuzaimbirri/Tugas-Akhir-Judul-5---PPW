# Kalkulator Web - Tugas Akhir 5 PPW

## Tentang Project

Ini adalah aplikasi kalkulator yang dibuat untuk memenuhi tugas akhir Pemrograman Web. Kalkulator ini dibangun dengan HTML5, CSS (Tailwind), dan JavaScript.
## Fitur Utama

### Operasi Dasar
- Penjumlahan, pengurangan, perkalian, dan pembagian
- Mendukung operasi berantai 
- Perhitungan mengikuti aturan PEMDAS

### Memory System
- **MC** - Hapus memori
- **MR** - Panggil nilai dari memori
- **M+** - Tambahkan angka ke memori
- **Mâˆ’** - Kurangi angka dari memori
- Indikator visual saat ada data tersimpan di memori

### History
- Menyimpan 5 perhitungan terakhir
- Klik item history untuk memanggil hasilnya kembali
- Otomatis tersimpan selama sesi berlangsung

### Keyboard Support
- **0-9** - Input angka
- **.** - Desimal
- **+, -, *, /** - Operator
- **Enter** atau **=** - Hitung hasil
- **Backspace** - Hapus input (CE)
- **Escape** - Reset semua (C)
